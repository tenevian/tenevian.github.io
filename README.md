# 디지털 학습 자원과 수학 성취도 분석 대시보드

이 프로젝트는 디지털 학습 자원과 수학 성취도 간의 상관관계를 분석하고 시각화하는 대시보드입니다.

## 주요 기능

- 디지털 학습 자원 현황 분석
- 수학 성취도 분석
- 디지털 자원과 성취도 상관관계 분석
- 정책 효과 분석
- 지역별, 연도별 데이터 필터링

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Plotly.js
- Boxicons

## 프로젝트 구조

```
.
├── index.html          # 메인 대시보드 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── app.js          # 대시보드 기능 구현
└── data/
    ├── digital_resources/  # 디지털 자원 관련 데이터
    └── correlation/        # 상관관계 분석 데이터
```

## 사용 방법

1. 대시보드 접속
2. 상단 메뉴에서 원하는 분석 섹션 선택
3. 검색 및 필터 기능 활용
4. 데이터 시각화 확인
